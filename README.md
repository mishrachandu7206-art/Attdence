# Attendance App- By Anshul Kumar

Simple teacher/student attendance web app.

Setup

1. Copy `.env.example` to `.env` and set `MONGO_URI`.
```
MONGO_URI=mongodb://localhost:27017/attendance_app
PORT=3000

```
3. Install:

```
npm install
```

3. Start server: copy and paste in terminal

```
npm run dev 
```

Open `http://localhost:3000/teacher._loginhtml` and `http://localhost:3000/student.html`.
